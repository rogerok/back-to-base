```ts
function demo<T extends { id: unknown }>(u: T): T["id"] extends string ? "str" : "num";
{
  if (typeof u.id === "string") {
    return "str" as T["id"] extends string ? "str" : "num";
  }

  return "num" as T["id"] extends string ? "str" : "num";
}
```

В таких случаях допустимо использовать `as` т.к. TypeScript не может вычислить типов возвращений в функциях, которые используют дженерик.
