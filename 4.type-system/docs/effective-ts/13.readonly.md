```ts
interface User {
  id: string;
  name: string;
}

const user: Readonly<User> = {
  id: "id",
  name: "John",
};

const process = (u: User) => {
  u.id = "123";
};

process(user); // Ошибка не возникает
```

`Readonly` защищает только по месту его использования, также в сигнатуре ф-ции идентфикатору `u` мы присвоили тип, отличный от `Readonly<User>` - `User`

`Readonly` является shallow, т.е. не работает на вложенные объекты.
Так же работает `readonly ` с массивами

## НО

В этом случае `as const` работает как глубокий `Readonly`

```ts
const foo = {
  a: 1,
  b: 2,
  c: {
    e: 3,
  },
} as const;
```
