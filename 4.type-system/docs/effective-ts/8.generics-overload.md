# Generics and Overloads

1) Дженерики это переменные для типов
2) `extends` (type constrains) описывают какие операции допустимы со значением внутри нашей ф-ции
3) Для вывода дженерика в функциях, используется только объявление ф-ции

```ts
function pick<Obj extends object, Key extends keyof Obj>(obj: Obj, key: Key) {
  return obj[key];
}
```

infer
---

```ts
type Ref<Value> = { value: Value }
type MaubeRefOrCallable<Value> = Value | Ref<Value> | (() => Value)

function useSomething(option): MaybeRefOrCallable<{ some: 1, weird: string, type: number }>

const foo = useSomething()
```

Для того чтобы извлечь возвращаемый тип из useSomething, можно использовать infer

```ts
type ExtractValue<T> = T extends MaybeRefOrCallable<infer Value> ? 'true' : 'false'
// Если мой тип T расширяет MaybeRefOrCallable, то сохрани в переменную Value первый аргумент этого дженерика
```

И теперь мы можем написать

```ts
type ExtractValue<T extends MaybeRefOrCallable<unknown>> = T extends MaybeRefOrCallable<infer Value> ? Value : never;
```
