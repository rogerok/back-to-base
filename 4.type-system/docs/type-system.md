# Что такое тип?

**Тип - это классификация данных, которая определяет**:

- Множество допустимых значений (Int, Bool etc)
- Множество допустимых операций над этими значениями (арифметика, конкатенация)

**Type system** - это набор правил, по которым язык программирования назначает типы различными конструкциям программы (
выражениям, переменным, функциям и тд)

**Система типов решает несколько задач**:

- Safety - предотвращение определенных классов ошибок (type safety гарантирует что well typed programs don't go wrong)
- Абстракция - скрытие деталей реализации за интерфейсом типа
- Документация - сигнатуры типов служат машинно-проверяемой документацией
- Оптимизация - компилятор может использовать информацию о типах для генерации более эффективного кода
