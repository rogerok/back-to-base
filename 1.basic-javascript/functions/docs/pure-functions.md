# Теория: Чистые функции

## Детерминированность

*Недетерминированная функция* - функция, которая при передаче в нее одних и тех же входных данных, может возвращать
разные значения.
Например `Math.randon()`, `Date.now()`.

Пример с аргументами:

```js
 getAge(2000)
```

Хотя прямо сейчас запуск вернет точно такое же значение, через год оно уже будет другим. Т.е. функция считается
недетерминированной, если так она себя ведет хотя бы единожды

Детерменированные ф-ции ведут себя предсказуемо - для одних и тех же входных данных они выдают один и тот же результат.

Ф-ция становится недетерменированной, если она обращается к некоторым внешним данным, например глобальным переменным,
переменным окружения и т.д.
Внешние данные могут измениться и результат выполнения ф-ции тоже будет отличаться.

## Побочные эффекты (side effects)

Вторая ключевая характеристка ф-ций - наличие побочных эффектов.
Побочными эффектами называются любые взаимодействия с внешней средой - файловые операции, запись в файл, чтение файла,
вывод в консоль, запрос данных по сети, изменение внешних переменных

Идеальная функция с точки зрения удобства работы с ней называется чистой (pure). Чистая функция — это детерминированная
функция, которая не производит побочных эффектов. Такая функция зависит только от своих входных аргументов и всегда
ведёт себя предсказуемо.
