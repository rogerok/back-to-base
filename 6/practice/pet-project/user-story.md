# Приложение для ведения личной базы знаний по обучению и практике.

Сервис для управления обучением разработчика. Пользователь создает навыки и темы, прикрепляет к ним учебные материалы,
ведет заметки, отслеживает прогресс и возвращается к темам, которые нужно повторить.

Ключевые возможности:
пользователь регистрируется и входит в аккаунт;
создает навыки, например TypeScript, React, SQL;
добавляет темы и подтемы внутри навыков;
прикрепляет к темам статьи, видео, курсы, документацию;
пишет заметки и конспекты по каждой теме;
отмечает статус темы и уровень своего понимания;
ставит цели по обучению и связывает их с темами;
видит dashboard с прогрессом, активными темами и темами на повторение;
использует поиск, фильтры и теги для быстрого досту

## Авторизация

1) Как пользователь, я хочу создать аккаунт и войти в него
    - Пользователь регистрируется с помощью электронной почты и пароля
    - Пользователь вводит и подтверждает пароль
    - Пользователь входит в приложение с помощью электронной почты и пароля
2) Как пользователь, я хочу иметь возможность сбросить пароль, чтобы войти в приложение, если я его забыл.
    - Пользователь может сбросить пароль, перейди по ссылке из письма, отправленного на электронную почту
    - Пользователь должен ввести и подтвердить пароль
3) Как пользователь, я хочу управлять навыками, чтобы наполнять их материалами.
    - Пользователю доступен CRUD функционал навыков
    - Пользователь задает название, описание, теги, приоритет навыка
4) Как пользователь, я хочу создавать темы для обучения
    - Пользователь создает темы для обучения в конкретном навыке
    - Пользователь задает название навыка, прогресс, уровень понимания, дедлайн
5) Как пользователь, я хочу видеть список своих тем в навыках
    - Пользователю доступен экран списка тем внутри навыка
6) Как пользователь, я хочу фильтровать темы, чтобы получить быстрый доступ к ним
    - Пользователь может фильтровать темы по дате создания, уровню понимания, прогрессу, заголовку
