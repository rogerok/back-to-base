# Notion Analog

## Описание

Персональная база знаний с иерархией страниц, блочным редактором и быстрым поиском. Упрощённый аналог Notion — без баз данных, но с удобной навигацией и гибким редактором.

---

## Экраны

### 1. Регистрация / Вход

- Форма регистрации: имя, email, пароль
- Форма входа: email, пароль

### 2. Основной layout (после входа)

- **Левый сайдбар:**
  - Имя пользователя / аватар
  - Кнопка быстрого поиска (Ctrl+K)
  - Кнопка "Создать страницу"
  - Дерево страниц с раскрытием вложенных
  - Раздел "Удалённые страницы" (Trash)
  - Настройки
- **Основная область:** содержимое выбранной страницы

### 3. Страница (основной экран)

- Поле заголовка страницы (большой, вверху)
- Выбор иконки страницы (emoji)
- Опциональное обложка страницы (цвет или картинка)
- Блочный редактор (см. блоки ниже)
- Хлебные крошки (путь в иерархии)
- Кнопка "..." (меню страницы): переименовать, переместить, удалить, копировать ссылку
- Дата последнего изменения

### 4. Блочный редактор

Поддерживаемые типы блоков:

| Блок | Описание |
|---|---|
| Текст | Обычный параграф |
| Заголовок H1 / H2 / H3 | Три уровня заголовков |
| Маркированный список | Bullet list |
| Нумерованный список | Numbered list |
| Чеклист | Список с чекбоксами |
| Цитата | Блок с вертикальной чертой слева |
| Разделитель | Горизонтальная линия |
| Код | Блок кода с выбором языка и подсветкой |
| Изображение | Загрузка или вставка по URL |
| Ссылка на страницу | Внутренняя ссылка на другую страницу |
| Callout | Выделенный блок с иконкой (для заметок/предупреждений) |

Поведение редактора:
- `/` в начале строки открывает меню выбора типа блока
- Enter создаёт новый блок того же типа
- Backspace на пустом блоке удаляет его и возвращает курсор выше
- Drag & drop для перестановки блоков
- Выделение текста показывает тулбар (жирный, курсив, подчёркнутый, зачёркнутый, код, ссылка)

### 5. Быстрый поиск (Ctrl+K / Cmd+K)

- Модальное окно поверх страницы
- Поиск по названиям страниц и содержимому блоков
- Результаты в реальном времени при наборе
- Переход на страницу по Enter или клику
- Клавиатурная навигация по результатам (стрелки)

### 6. Trash (удалённые страницы)

- Список удалённых страниц с датой удаления
- Кнопки: восстановить, удалить окончательно
- Очистить всё

### 7. Настройки

- Имя, email
- Аватар (загрузка фото)
- Смена пароля
- Тема: светлая / тёмная / системная
- Удаление аккаунта

## Функционал

### Страницы
- Создание, переименование, удаление (в корзину)
- Иерархия: страница может содержать вложенные страницы (неограниченная вложенность)
- Drag & drop страниц в сайдбаре для перемещения
- Иконка (emoji) и обложка для каждой страницы
- Копирование ссылки на страницу

### Блочный редактор
- 11 типов блоков
- Slash-команды (`/`) для вставки блоков
- Drag & drop блоков
- Инлайн-форматирование текста (bold, italic, underline, strikethrough, inline code, link)
- Автосохранение при изменениях (debounce)

### Поиск
- Полнотекстовый поиск по заголовкам и содержимому блоков
- Быстрый доступ через Ctrl+K
- Клавиатурная навигация

### Навигация
- Дерево страниц в сайдбаре с раскрытием/схлопыванием
- Хлебные крошки на странице
- Быстрый переход через поиск

### Корзина
- Мягкое удаление страниц (soft delete)
- Восстановление из корзины
- Окончательное удаление

### Темы
- Светлая / тёмная / системная тема

---

## Технические особенности

- Контент страницы хранится как массив блоков (JSON) в БД
- Автосохранение через debounce (500мс после последнего изменения)
- Полнотекстовый поиск на бэке
- Иерархия страниц через рекурсивные запросы (adjacency list или nested sets)
- Валидация блоков через zod (тип + payload для каждого типа блока)
- Drag & drop: библиотека `dnd-kit` на фронте
