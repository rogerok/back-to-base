# Finance Tracker

## Описание

Личный трекер финансов. Записываешь доходы и расходы, раскидываешь по категориям, ставишь бюджеты на месяц и видишь куда уходят деньги через дашборд с графиками.

---

## Экраны

### 1. Регистрация / Вход

- Форма регистрации: имя, email, пароль, валюта по умолчанию
- Форма входа: email, пароль
- Валидация в реальном времени

### 2. Дашборд

- Баланс за текущий месяц (доходы - расходы)
- Виджет "Расходы по категориям" — круговая диаграмма
- Виджет "Доходы vs расходы" — столбчатый график по неделям
- Виджет "Топ категорий расходов" за месяц
- Виджет "Прогресс бюджетов" — список активных бюджетов с прогресс-барами
- Последние 5 транзакций
- Переключатель периода: текущий месяц / прошлый месяц / произвольный диапазон

### 3. Транзакции

- Список всех транзакций с пагинацией
- Каждая строка: дата, категория (иконка + цвет), описание, сумма (красная/зелёная)
- Поиск по описанию
- Фильтры: тип (доход/расход), категория, период, диапазон сумм
- Сортировка по дате / сумме
- Кнопка "Добавить транзакцию"
- Клик на транзакцию — детальный просмотр / редактирование

### 4. Добавление / редактирование транзакции

- Переключатель: расход / доход
- Поле: сумма
- Выбор категории (с иконкой и цветом)
- Поле: описание (опционально)
- Поле: дата (по умолчанию сегодня)
- Поле: заметка (опционально)
- Кнопки: сохранить, отмена

### 5. Категории

- Список категорий разбитый на две группы: расходы и доходы
- Каждая категория: иконка, цвет, название, количество транзакций за месяц, сумма за месяц
- Кнопка "Создать категорию"
- Редактирование и удаление (с предупреждением если есть транзакции)
- Дефолтные категории при регистрации: Еда, Транспорт, Развлечения, Здоровье, Одежда, Коммунальные, Зарплата, Фриланс

### 6. Создание / редактирование категории

- Поле: название
- Тип: расход / доход
- Выбор иконки (из набора)
- Выбор цвета

### 7. Бюджеты

- Список активных бюджетов текущего месяца
- Каждый бюджет: категория, лимит, потрачено, остаток, прогресс-бар (меняет цвет при приближении к лимиту)
- Статус: в норме / предупреждение (>80%) / превышен
- Кнопка "Добавить бюджет"
- История бюджетов прошлых месяцев

### 8. Создание / редактирование бюджета

- Выбор категории расходов
- Поле: лимит суммы
- Поле: месяц (по умолчанию текущий)
- Опция: повторять каждый месяц автоматически

### 9. Отчёты

- Выбор периода: месяц, квартал, год, произвольный
- Таблица расходов по категориям за период
- График динамики расходов по месяцам
- Сравнение двух периодов
- Экспорт в CSV

### 10. Настройки

- Имя, email
- Валюта по умолчанию
- Смена пароля
- Управление категориями (ссылка)
- Удаление аккаунта

---

## Функционал

### Аутентификация
- Регистрация / вход по email + пароль
- JWT + refresh токены

### Транзакции
- Типы: расход / доход
- Привязка к категории
- Поиск и фильтрация
- Редактирование и удаление

### Категории
- Пользовательские категории (иконка + цвет + тип)
- Дефолтный набор при регистрации
- Нельзя удалить если есть транзакции (или с переносом на другую категорию)

### Бюджеты
- Лимит расходов по категории на месяц
- Автоматический пересчёт остатка при новых транзакциях
- Повторение бюджетов каждый месяц
- Предупреждение при превышении 80% лимита

### Аналитика
- Расходы / доходы по категориям за период
- Динамика по неделям и месяцам
- Сравнение периодов

### Экспорт
- Транзакции в CSV за выбранный период

---

## Технические особенности

- Валидация через zod (суммы, даты, типы)
- Агрегации и группировки на бэке (Fastify + SQL)
- Хранение сумм в целых числах (копейки/центы) во избежание проблем с float
- fp-ts для трансформаций данных на бэке
