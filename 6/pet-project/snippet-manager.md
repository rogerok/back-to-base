# Snippet Manager

## Описание

Персональный менеджер сниппетов кода. Сохраняешь полезные куски кода, организуешь по тегам и языкам, быстро находишь через поиск и копируешь одной кнопкой.

---

## Экраны

### 1. Регистрация / Вход

- Форма регистрации: имя, email, пароль
- Форма входа: email, пароль
- Валидация полей в реальном времени

### 2. Главная — список сниппетов

- Сетка или список сниппетов (переключатель вид)
- Каждая карточка: название, язык (бейдж с цветом), теги, превью первых строк кода, дата создания
- Строка поиска (по названию, описанию и содержимому кода)
- Фильтр по языку (мультиселект)
- Фильтр по тегам (мультиселект)
- Сортировка: по дате создания, по дате изменения, по названию
- Кнопка "Создать сниппет"
- Счётчик найденных результатов

### 3. Просмотр сниппета

- Название
- Язык программирования (бейдж)
- Теги
- Описание (если есть)
- Блок кода с подсветкой синтаксиса
- Кнопка "Копировать код" (с анимацией подтверждения)
- Дата создания и последнего изменения
- Кнопки: редактировать, удалить
- Раздел "Похожие сниппеты" (по тегам / языку)

### 4. Создание сниппета

- Поле: название (обязательно)
- Выбор языка программирования (searchable select, ~30 популярных языков)
- Редактор кода с подсветкой синтаксиса
- Поле: описание (опционально, markdown)
- Поле: теги (мультиселект с автодополнением + создание нового тега)
- Предпросмотр (переключатель редактор / превью для описания)
- Кнопки: сохранить, отмена

### 5. Редактирование сниппета

- Те же поля что при создании, заполненные текущими значениями
- Кнопки: сохранить изменения, отмена, удалить

### 6. Теги — боковая панель / отдельный экран

- Список всех тегов
- Рядом с каждым тегом: количество сниппетов
- Клик на тег — фильтрует список сниппетов
- Переименование тега (инлайн)
- Удаление тега с предупреждением (тег снимается со всех сниппетов)

### 7. Коллекции (папки)

- Список коллекций в сайдбаре: название, количество сниппетов
- Создание коллекции: название
- Добавление сниппета в коллекцию через меню на карточке
- Сниппет может быть в нескольких коллекциях
- Фильтрация по коллекции
- Переименование и удаление коллекции

### 8. Настройки

- Имя, email
- Смена пароля
- Тема редактора кода (VS Code Dark, GitHub Light, etc.)
- Размер шрифта в редакторе
- Экспорт всех сниппетов в JSON
- Импорт из JSON
- Удаление аккаунта

---

## Функционал

### Аутентификация
- Регистрация / вход по email + пароль
- JWT + refresh токены

### Сниппеты
- CRUD: создание, чтение, редактирование, удаление
- Поддержка ~30 языков программирования (JS, TS, Python, Go, Rust, SQL, Bash, etc.)
- Подсветка синтаксиса в редакторе и при просмотре
- Копирование кода одной кнопкой
- Markdown-описание с превью
- Мягкое удаление (soft delete) с возможностью восстановления

### Поиск
- Полнотекстовый поиск по названию, описанию и содержимому кода
- Поиск в реальном времени (debounce)
- Комбинированная фильтрация: язык + теги + коллекция

### Теги
- Создание при добавлении сниппета
- Автодополнение существующих тегов
- Переименование и удаление тегов
- Кликабельные теги в списке и на странице сниппета

### Коллекции
- Группировка сниппетов в именованные папки
- Сниппет может быть в нескольких коллекциях
- Сортировка внутри коллекции

### Импорт / Экспорт
- Экспорт всех сниппетов в JSON
- Импорт из JSON (с валидацией и дедупликацией)

---

## Технические особенности

- Подсветка синтаксиса: библиотека `shiki` (работает на сервере и клиенте)
- Полнотекстовый поиск на бэке через SQL `LIKE` или `pg_trgm` (PostgreSQL)
- Валидация через zod (язык из enum, теги — массив строк, etc.)
- Дебаунс поиска на фронте (300мс)
- Редактор кода: `CodeMirror` или `Monaco Editor`
