# Учебные материалы: реализация роутера

## Как пользоваться

Материалы расположены в порядке возрастания "подсказывания". Выбери свой путь:

**Путь A (максимум самостоятельности):** 01 → 04 → 03 (только если застрял)
**Путь B (с псевдокодом):** 01 → 02 → 03

| # | Файл | Что внутри | Когда читать |
|---|------|------------|--------------|
| 1 | [01-concepts-and-theory.md](./01-concepts-and-theory.md) | Теория, ментальные модели, архитектура | Перед началом — чтобы понять ЧТО и ЗАЧЕМ |
| 2 | [02-step-by-step-guide.md](./02-step-by-step-guide.md) | Пошаговый план, псевдокод, подсказки | Во время реализации — чтобы знать КАК |
| 3 | [03-reference-implementation.md](./03-reference-implementation.md) | Готовый код по секциям | Когда застрял — заглядывай только в нужную секцию |
| 4 | [04-implementation-guide.md](./04-implementation-guide.md) | Пошаговый гайд **без кода** — три уровня подсказок | Альтернатива файлу 02 для самостоятельной работы |

## Рекомендуемый порядок работы

1. Прочитай теорию (файл 01) целиком
2. Открой файл 04 (без кода) или 02 (с псевдокодом) — на выбор
3. Реализуй компоненты по шагам, открывая подсказки только когда застрял
4. Если совсем застрял на конкретном шаге — загляни в соответствующую секцию файла 03
5. Прогоняй тесты ментора после каждого крупного шага
